# Gia Binh Smart Airport – Feedback V3

Các cập nhật chính trong bản này:

1. **Overview 2D**
   - Dời cụm nút `Show data layers / Operational overview / Operational insights` sang phải để tránh đè lên `Airport Overview`.

2. **Overview 3D**
   - Nới giới hạn zoom để zoom linh hoạt hơn.
   - Bổ sung **2 chế độ điều khiển**:
     - `Orbit`
     - `Walk` (free-walk / unphysical) với hướng dẫn thao tác.
   - Tăng độ gần của camera mặc định khi vào chế độ 3D để mô hình nhìn lớn hơn.
   - Nâng cấp nền không gian đêm theo hướng HDRI giả lập với dome bầu trời, sao và ánh sáng tốt hơn.
   - Làm lại `grid floor` với lưới dày hơn, nhiều ô nhỏ và cảm giác 3D hơn.
   - Bổ sung platform hiển thị dưới mô hình để mô hình đứng đẹp hơn, hạn chế cảm giác bóng bị “bay”.
   - Cải thiện bóng đổ và cảm giác neo mô hình xuống mặt hiển thị.

## Cách chạy
- Nháy đúp `OPEN_GIA_BINH.cmd`
- Hoặc chạy:
  - `npm install`
  - `npm run dev`
