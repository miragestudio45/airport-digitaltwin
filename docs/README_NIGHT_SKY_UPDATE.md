# Night Sky Update

The Overview 3D sky has been rebuilt based on the visual direction of `bautrodem(1).html`:

- Deep navy-to-black atmospheric gradient
- Soft blue horizon airglow
- Deterministic tiny star field (no random flicker between reloads)
- Milky Way star band with subtle warm galactic core
- Soft, low-opacity galactic cloud layer
- Very slow sidereal rotation only; no brightness twinkle
- Stars respect scene depth, so they do not render through the airport model

No other functional module was intentionally changed in this update.
